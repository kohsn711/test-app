# 18 最低限のテスト

## 概要
MVPリリース前の動作確認チェックリスト。自動テストは未設定のため手動で確認。

## ToDo

### 認証フロー
- [ ] 新規ユーザーがメールアドレスでログインできる
- [ ] 認証コードで認証できる
- [ ] ログアウトできる
- [ ] 未認証ユーザーがログインページへリダイレクトされる

### 学生フロー
- [ ] 初期設定が完了できる
- [ ] チームコードでチームに参加できる
- [ ] 今日の記録を作成できる（全カテゴリ）
- [ ] 記録を編集できる
- [ ] 過去の記録を閲覧できる
- [ ] カレンダーに記録済み日が表示される
- [ ] 連続記録日数が正しく表示される
- [ ] 通知が届く

### コーチフロー
- [ ] チームを作成できる
- [ ] 所属チームの選手一覧が表示される
- [ ] 選手の記録を閲覧できる（他チームの選手は閲覧できない）
- [ ] リアクションを送れる
- [ ] 定型コメントを送れる

### 保護者フロー
- [ ] 保護者承認ができる
- [ ] 紐づく子どもの記録を閲覧できる（紐づかない学生は閲覧できない）
- [ ] リアクションを送れる
- [ ] 定型コメントを送れる

### 管理者フロー
- [ ] コンテンツを作成できる
- [ ] コンテンツを公開できる
- [ ] 公開コンテンツが学生・保護者に表示される

### 権限チェック
- [ ] studentが他人の記録にアクセスできない
- [ ] coachが他チームの選手記録にアクセスできない
- [ ] parentが紐づかない子どもの記録にアクセスできない
- [ ] 未認証ユーザーがすべての保護ルートにアクセスできない

### ビルド確認
- [ ] `npm run build` がエラーなく完了する
- [ ] `npm run lint` がエラーなく完了する
