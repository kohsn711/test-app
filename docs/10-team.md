# 10 チーム作成・チーム参加

## 概要
コーチがチームを作成し、学生がチームコードで参加する機能。

## ToDo

### チーム作成（コーチ）
- [ ] `app/(coach)/team/create/page.tsx` を作成
- [ ] チーム名入力フォーム
- [ ] 作成Server Action（`teams` テーブルに insert、`team_members` にcoachとして登録）
- [ ] チームコード（ランダム英数字6〜8文字）を自動生成して保存

### チーム参加（学生）
- [ ] `app/(student)/team/join/page.tsx` を作成
- [ ] チームコード入力フォーム
- [ ] 参加Server Action（コードでチームを検索し `team_members` にstudentとして登録）
- [ ] 存在しないコードの場合はエラーメッセージを表示

### チーム情報表示
- [ ] コーチのダッシュボードにチームコードを表示
- [ ] 学生のマイページに所属チーム名を表示

### アクセス制御
- [ ] チーム作成はcoachロールのみ
- [ ] チーム参加はstudentロールのみ
- [ ] 1学生が複数チームに所属できる設計（または制限する場合はその旨をコードに明示）

## 備考
- チームコードは推測されにくいランダム文字列にする
